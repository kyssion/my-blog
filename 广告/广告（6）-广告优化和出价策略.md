
# 📋 广告出价与优化策略完整全景图 (2026 版)

## 一、竞价机制与拍卖类型 (Auction Mechanisms & Types)

| 术语 | 定义 | 说明 | 应用场景 |
|------|------|------|---------|
| **First Price Auction (GFP)** | 第一价格拍卖 | 最高出价者按自己出价支付 | 程序化广告、Header Bidding |
| **Second Price Auction (GSP)** | 第二价格拍卖 | 最高出价者按次高出价 +0.01 支付 | Google 搜索广告 |
| **VCG Auction** | VCG 拍卖机制 | 按社会成本计费，鼓励真实出价 | Facebook 部分场景 |
| **Generalized First Price (GFP)** | 广义第一价格 | 第一价格拍卖的变体 | 部分程序化平台 |
| **Generalized Second Price (GSP)** | 广义第二价格 | 第二价格拍卖的变体，考虑质量分 | Google/Bing 搜索 |
| **Expressive Auction** | 表达性拍卖 | 允许广告主提交复杂出价表达式 | 高级程序化 |
| **Combinatorial Auction** | 组合拍卖 | 允许对广告位组合出价 | 多位置打包 |
| **Sequential Auction** | 序列拍卖 | 多个广告位依次拍卖 | 多广告位场景 |
| **Simultaneous Auction** | 同时拍卖 | 多个广告位同时拍卖 | 多广告位场景 |
| **RTB (Real-Time Bidding)** | 实时竞价 | 毫秒级程序化竞价，每次展示独立拍卖 | 程序化展示广告 |
| **Header Bidding** | 头部竞价 | 让多个广告交易所同时竞价 | 媒体方收益优化 |
| **Unified Auction** | 统一拍卖 | Google Ad Manager 的统一竞价机制 | 谷歌广告生态 |
| **Floor Price Strategy** | 底价策略 | 设置最低可接受出价，过滤低质流量 | 媒体方/SSP |
| **Dynamic Floor Price** | 动态底价 | 根据流量质量动态调整底价 | 程序化交易 |
| **Hard Floor** | 硬底价 | 绝对不可低于的最低价格 | 媒体方保护 |
| **Soft Floor** | 软底价 | 可协商的最低价格参考 | 媒体方优化 |
| **Bid Ceiling** | 出价上限 | 设置最高出价限制，防止超预算 | 广告主侧 |
| **Bid Floor** | 出价底价 | 程序化交易中的最低出价门槛 | DSP/ADX |
| **Reserve Price** | 保留价格 | 媒体方设置的最低成交价格 | SSP/Ad Exchange |
| **Spot Market** | 现货市场 | 即时交易的广告库存市场 | 程序化交易 |
| **Forward Market** | 远期市场 | 提前锁定未来广告库存 | 品牌广告 |

## 二、基础计费模式 (Basic Pricing Models)

| 术语 | 定义 | 说明 | 应用场景 |
|------|------|------|---------|
| **CPM (Cost Per Mille)** | 千次展示成本 | 品牌曝光导向 | 品牌广告、展示广告 |
| **vCPM (Viewable CPM)** | 可见千次展示成本 | 仅计算可见展示 | 品牌广告 |
| **CPC (Cost Per Click)** | 单次点击成本 | 流量导向 | 搜索广告、信息流 |
| **CPA (Cost Per Action)** | 单次转化成本 | 效果导向 | 效果广告 |
| **CPV (Cost Per View)** | 单次观看成本 | 视频广告专用 | YouTube、抖音视频 |
| **CPV (Cost Per Visit)** | 单次访问成本 | 网站访问导向 | 引流广告 |
| **CPL (Cost Per Lead)** | 单次线索成本 | 表单/留资导向 | B2B、教育、金融 |
| **CPS (Cost Per Sale)** | 单次销售成本 | 电商成交导向 | 电商广告、联盟营销 |
| **CPI (Cost Per Install)** | 单次安装成本 | 应用下载导向 | APP 推广 |
| **CPH (Cost Per Hour)** | 单次通话成本 | 电话转化导向 | 本地服务、电话咨询 |
| **CPE (Cost Per Engagement)** | 单次互动成本 | 社交互动导向 | 社交媒体广告 |
| **CPCV (Cost Per Completed View)** | 单次完整观看成本 | 视频完播导向 | 视频广告 |
| **CPD (Cost Per Day)** | 单日成本 | 包天广告 | 开屏广告、首页位 |
| **CPW (Cost Per Week)** | 单周成本 | 包周广告 | 长期展示位 |
| **CPM+ (CPM Plus)** | CPM 加分成 | 基础 CPM 加收入分成 | 混合计费 |
| **Revenue Share** | 收入分成 | 按收入比例分成 | 联盟营销、网红合作 |
| **Hybrid Pricing** | 混合计费 | 多种计费模式组合 | 复杂场景 |
| **Performance-Based** | 效果计费 | 基于实际效果付费 | 效果广告 |
| **Flat Fee** | 固定费用 | 固定金额购买广告位 | 品牌广告 |
| **Cost Per Conversation** | 单次对话成本 | 聊天/客服对话导向 | 客服广告 |

| **Deep Conversion Bidding** | 深度转化出价 | 基于深度转化事件优化出价 | 巨量引擎 |
| **Dual-Objective Bidding** | 双目标出价 | 同时优化两个目标 (如转化 +ROAS) | 复杂目标 |

| **Margin-Based Bidding** | 利润率出价 | 基于产品利润率差异化出价 | 电商 |
| **Inventory-Based Bidding** | 库存驱动出价 | 基于库存水平调整出价 | 电商清仓 |

## 四、出价控制技术 (Bid Control Techniques)

| 术语 | 定义 | 说明 | 应用场景 |
|------|------|------|---------|
| **Bid Shading** | 出价遮蔽 | 在第二价格拍卖中智能降低出价，略高于次高出价即可获胜 | 程序化广告、DSP |
| **Bid Multiplier** | 出价系数/出价调节器 | 对特定维度 (时段/地域/设备/人群) 设置±% 出价调整 | 所有主流平台 |
| **Bid Adjustment** | 出价调整 | 广义的出价系数调整 | 通用术语 |
| **Device Bid Adjustment** | 设备出价调整 | 针对手机/平板/电脑差异化出价 | 多设备投放 |
| **Location Bid Adjustment** | 地域出价调整 | 针对不同地理位置调整出价 | 本地/全国投放 |
| **Time Bid Adjustment** | 时段出价调整 | 针对不同时间段调整出价 | Dayparting |
| **Audience Bid Adjustment** | 人群出价调整 | 针对不同人群包调整出价 | 精细化定向 |
| **Browser/OS Bid Adjustment** | 浏览器/系统出价调整 | 针对特定浏览器或操作系统 | 技术定向 |
| **Connection Type Adjustment** | 网络类型调整 | WiFi/4G/5G 等不同网络环境 | 移动端优化 |
| **Placement Bid Adjustment** | 版位出价调整 | 针对不同展示位置调整出价 | 展示广告 |
| **Keyword Match Type Adjustment** | 关键词匹配类型调整 | 针对不同匹配类型调整出价 | 搜索广告 |
| **Ad Schedule Adjustment** | 广告时间表调整 | 按小时/星期调整出价 | 时段优化 |
| **Demographic Adjustment** | 人口统计调整 | 按年龄/性别/收入调整出价 | 人群定向 |
| **Weather-Based Adjustment** | 天气驱动调整 | 根据天气条件调整出价 | 本地服务 |
| **Event-Based Adjustment** | 事件驱动调整 | 根据特殊事件调整出价 | 营销活动 |
| **Competitor-Based Adjustment** | 竞争驱动调整 | 根据竞争对手活动调整出价 | 竞争策略 |
| **Real-Time Bid Adjustment** | 实时出价调整 | 基于实时信号动态调整 | 程序化广告 |
| **Predictive Bid Adjustment** | 预测性出价调整 | 基于预测模型调整出价 | AI 优化 |
| **Constraint-Based Bidding** | 约束出价 | 在多重约束条件下优化出价 | 复杂场景 |
| **Risk-Adjusted Bidding** | 风险调整出价 | 考虑风险因素的出价策略 | 保守策略 |

## 五、预算与消耗控制 (Budget & Pacing Control)

| 术语 | 定义 | 说明 | 应用场景 |
|------|------|------|---------|
| **Budget Pacing** | 预算平滑 | 控制预算在周期内均匀/智能消耗 | 所有广告投放 |
| **Standard Pacing** | 标准平滑 | 均匀分配预算到每个时间段 | 稳定投放需求 |
| **Accelerated Pacing** | 加速消耗 | 尽快花完预算，抢占流量 | 促销/紧急 campaign |
| **Predictive Pacing** | 预测性平滑 | 基于历史数据预测最优消耗节奏 | 智能投放系统 |
| **Adaptive Pacing** | 自适应平滑 | 根据实时表现动态调整消耗节奏 | 智能优化 |
| **End-of-Day Rush** | 日末冲刺 | 避免预算在日末未花完的加速策略 | 每日预算场景 |
| **Front-Loading** | 前置消耗 | 在周期前期集中消耗预算 | 新品发布 |
| **Back-Loading** | 后置消耗 | 在周期后期集中消耗预算 | 促销收尾 |
| **CBO (Campaign Budget Optimization)** | 广告系列预算优化 | 在广告组间动态分配预算 | Meta、Google |
| **ABO (Ad Set Budget Optimization)** | 广告组预算优化 | 手动分配各广告组预算 | Meta 手动控制 |
| **Daily Budget** | 每日预算 | 单日预算上限 | 常规投放 |
| **Lifetime Budget** | 生命周期预算 | 整个广告周期总预算 | 短期活动 |
| **Shared Budget** | 共享预算 | 多个广告系列共用预算池 | 多计划协同 |
| **Budget Reallocation** | 预算重新分配 | 根据表现动态转移预算 | 优化阶段 |
| **Seasonal Budget Adjustment** | 季节性预算调整 | 根据促销/节日临时调整预算 | 电商大促 |
| **Budget Smoothing** | 预算平滑 | 避免预算消耗波动过大 | 稳定投放 |
| **Budget Guardrails** | 预算护栏 | 设置预算上下限保护 | 风险控制 |
| **Spend Cap** | 花费上限 | 设置总花费绝对上限 | 预算保护 |
| **Minimum Spend** | 最低花费 | 确保预算最低消耗 | 合约广告 |
| **Budget Flexibility** | 预算灵活性 | 允许预算在一定范围内浮动 | 智能投放 |
| **Cross-Campaign Budget** | 跨系列预算 | 多个广告系列共享预算 | 整合营销 |
| **Rolling Budget** | 滚动预算 | 未使用预算可滚入下一周期 | 长期投放 |

## 六、频控与曝光管理 (Frequency & Exposure Management)

| 术语 | 定义 | 说明 | 应用场景 |
|------|------|------|---------|
| **Frequency Capping** | 频次控制 | 限制用户单位时间内的广告曝光次数 | 品牌广告、避免骚扰 |
| **Recency Capping** | 近期频控 | 限制特定时间窗口内的曝光次数，如"7 天最多 3 次" | 再营销、转化后用户 |
| **Global Frequency Cap** | 全局频控 | 跨广告系列/账户级别的频控 | 多计划协同投放 |
| **Per-User Frequency** | 单用户频控 | 针对单个用户的曝光限制 | 精准营销 |
| **Per-Session Frequency** | 单次会话频控 | 用户单次访问期间的曝光限制 | 移动端 APP |
| **Daily Frequency Cap** | 每日频控 | 单用户每日曝光上限 | 常规控制 |
| **Weekly Frequency Cap** | 每周频控 | 单用户每周曝光上限 | 长期投放 |
| **Monthly Frequency Cap** | 每月频控 | 单用户每月曝光上限 | 长期品牌 |
| **Lifetime Frequency Cap** | 生命周期频控 | 单用户整个活动周期曝光上限 | 短期活动 |
| **Reach vs Frequency Trade-off** | 触达与频次权衡 | 在覆盖更多用户 vs 重复曝光之间平衡 | 预算分配策略 |
| **Effective Frequency** | 有效频次 | 达成营销目标所需的最小曝光次数 | 媒体计划 |
| **Frequency Distribution** | 频次分布 | 分析不同曝光次数用户的占比 | 效果分析 |
| **Optimal Frequency** | 最优频次 | 通过测试找到的最佳曝光次数 | 优化目标 |
| **Frequency Weighting** | 频次加权 | 不同曝光次数赋予不同权重 | 归因分析 |
| **Cross-Platform Frequency** | 跨平台频控 | 多平台统一频控管理 | 全渠道营销 |
| **Sequential Frequency** | 序列频控 | 按用户旅程阶段设置不同频次 | 漏斗营销 |
| **Dynamic Frequency** | 动态频控 | 根据用户行为动态调整频次 | 智能优化 |
| **Negative Frequency** | 负向频控 | 转化后降低或停止曝光 | 效率优化 |
| **Incremental Frequency** | 增量频控 | 仅计算增量曝光的频次 | 增量分析 |
| **View-Through Frequency** | 浏览频控 | 基于浏览而非点击的频控 | 展示广告 |
| **Engagement Frequency** | 互动频控 | 基于用户互动的频控 | 社交广告 |

## 七、受众与定向策略 (Audience & Targeting Strategies)
